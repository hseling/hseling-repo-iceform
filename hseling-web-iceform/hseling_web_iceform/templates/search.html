Поиск по корпусу конструкций

Алгоритм опеределения значимых повторов основан на формировании списка нграмм и его фильтрации. Далее контркции объединяются в группы на основе семантической и контекстной схожести.

Настоящий поиск предоставляет возможность поиска по семантическим группам. Вы можете отфильтровать выдачу по суммарной частотности конструкций, входящих в группу, и количеству текстов, в которых эти конструкции встречаются. Каждая представленная в выдаче конструкция - один из примеров, который входит в группу. Нажатие на один из примеров позволит увидеть все констркции в группе и контексты, в которых они встречаются.

Выдача отфильтрована по первой букве глагола.

Для запуска нажмите кнопку "Поиск".
